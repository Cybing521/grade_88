## 📝 质量评价方法

### GRADE评价
- **High (⊕⊕⊕⊕)**：进一步研究不太可能改变效应估计的可信度
- **Moderate (⊕⊕⊕○)**：进一步研究可能对效应估计有重要影响
- **Low (⊕⊕○○)**：进一步研究很可能改变效应估计
- **Very Low (⊕○○○)**：效应估计非常不确定

### SYRCLE评价（动物实验）
采用10项偏倚风险评估条目，包括：
- 序列生成、基线特征、分配隐藏
- 随机安置、盲法实施
- 结局评估盲法、不完整数据处理等

### Cochrane ROB 2.0（临床研究）
评估5个偏倚风险领域：
- 随机化过程、偏离预定干预、结局数据缺失
- 结局测量、选择性报告

## ⚠️ 证据局限性

1. 大多数研究未报告分配隐藏和盲法实施情况
2. 研究间异质性较大（I² > 50%）
3. 部分结果未达到统计学显著性
4. 纳入研究数量有限
